## Install

```sh
yarn add @ovh-ux/ng-ovh-slider
```

## Usage

```js
import angular from 'angular';
import ngOvhSlider from '@ovh-ux/ng-ovh-slider';

angular
  .module('myModule', [
    ngOvhSlider,
  ]);
```

## Test

```sh
yarn test
```
