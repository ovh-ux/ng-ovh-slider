# Installation

## Bower

    bower install ovh-angular-slider --save

## NPM

    npm install ovh-angular-slider --save

## Get the sources

```bash
    git clone https://github.com/ovh-ux/ovh-angular-slider.git
    cd ovh-angular-slider
    npm install
    bower install
```
